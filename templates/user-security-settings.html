Multi-Factor Authentication (MFA)
Add an extra layer of security to your account. When enabled, you'll need to enter a verification code from your email in addition to your password.
Loading status...

How MFA Works:

  • After enabling MFA, you'll receive a 6-digit verification code via email on each login
  • You'll need to enter this code in addition to your password
  • Codes expire after 10 minutes for security
  • This prevents unauthorized access even if your password is compromised
